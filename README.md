## 📱 项目概述

CodeMate Mobile是一个专为移动设备设计的AI编程助手应用，集成了代码编辑、AI辅助、编译执行、版本控制等完整开发功能。

## 🚀 快速开始

### 环境要求
- **JDK**: 17+
- **Android SDK**: 34+
- **最低Android版本**: 6.0 (API 23)

### 克隆项目
```bash
git clone https://github.com/xzmaster54088/codemate-mobile.git
cd codemate-mobile
```

### 构建项目
```bash
./quick-build.sh debug
```

- **🖥️ 专业代码编辑器** - 14种语言语法高亮
